# Product Direction

## One-line summary
Thinking Canvas is a visual, structured, agent-operable thinking workspace for long-term idea growth.

## Core problem
Jones has ideas across conversations, repos, and working documents, but needs a system where those ideas can become visible, editable, and structurally manageable over time.

## Core bet
A tree-first visual interface with a flexible underlying structure is the right starting point.

## Core product loop
1. Capture a thought
2. Place it on a canvas
3. Expand or reorganize it
4. Ask Jarvis to operate on it
5. Preserve it as durable structure

## Why now
The interaction pattern is already validated by adjacent projects:
- IdeaCanvas validated editable visual node interaction
- trip-planner validated agent-writeable structured data
- thinking_with_ai validated long-term knowledge compounding behavior
